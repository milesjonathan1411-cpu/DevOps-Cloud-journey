EC2: stands for Elastic Compute Cloud. this instance is used to create a web server. This web server is used to automate deployments of any application.
Security group: whenever we deploy applications on a webserver, that application is going to run on aspecific port number (port number for webservers is 80). Port numbers are going to be defined on sercurity group.
TYPES OF SECURITY GROUP
======================
SSH: Secure Shell or Secure Socket Host with the port number 22. the main purpose of using SSH is to connect to the server

Key Pair: the main purpose of key pair is for logging in. 
TYPES OF KEY PAIR
=================
Public keypair
Private keypair

PUBLIC KEYPAIR: This type of keypair is managed by AWS.  

PRIVATE KEYPAIR: This keypair is managed bu the User. 

NOTE: Under private keypair, there are 2 types of private keypair named;
PEM and PPK.
